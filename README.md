# She Can Foundation Website

A responsive frontend website developed for **She Can Foundation** as part of an internship assignment. The website is designed to promote the foundation's mission of empowering women through education, healthcare awareness, skill development, and community engagement.

## Features

* Responsive landing page
* About Us section
* Mission, Vision, and Impact information
* Program pages:

  * Women Empowerment
  * Education
  * Healthcare Awareness
  * Skill Development
* Support Us / Donation page
* Volunteer registration form
* Modern UI with image-based hero sections
* GitHub Pages deployment

## Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages
